Carrier is a time-based puzzle game about encoding radio waves.

This code is publicly available for learning purposes, and can be borrowed, but is very specific to the type of game that Carrier is

Credit also given to Ryan Davis, and Justin Maynard as part of the Game Jam 2017 team.

Carrier is a Cuirass Entertainment Ltd. Game, and will be improved and released at a later date.
