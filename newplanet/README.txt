A New Planet is a completely procedurally generated game. It was written in object-oriented C#.

The use several different generation scripts to create a unique world using offset hex tiles, and then generate wildlife with unique
behavioural patterns, body parts, and colour schemes.

The Unity hierarchy is extremely clean for this project and features only one gameobject, the rest is handled entirely through code. The code also does a good job of hiding and parenting all the files so the hierarchy stays clean and organized while running. This makes it a clean and comfortable project to identify problems in
